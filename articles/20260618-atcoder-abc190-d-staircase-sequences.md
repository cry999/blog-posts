## TL;DR

- 連続する整数列（負の数も込み）の和が $N$ になるものを数える問題。素直に「$n$ を回して二分探索」だと $O(N \log N)$ で TLE。
- 和の式を整理すると $(n-m)(n+m+1) = 2N$。この 2 因数は偶奇が必ず異なるので、$2N$ を偶奇の違う 2 因数に分解する通り数を数えればよく、約数列挙 $O(\sqrt{N})$ で解ける。
- 結論は **答え＝$N$ の奇約数の個数 ×2**。しかもこの「×2」は、同じ和が「列を削る／負の項を足す」の二通りに読める二重性から来ている——そこが気持ちいい。

## この問題について

- ABC190 D - Staircase Sequences（公差 1 の等差数列＝連続整数列の和が $N$ になる数列の個数を数える）
- 問題ページ: https://atcoder.jp/contests/abc190/tasks/abc190_d
- この記事で伝えたいこと: 素直な「公式 → 二分探索」だと間に合わず、式をいじっていたら **約数列挙** に行き着いた、その道のりが面白かった話

## 問題概要

- 整数 $N$ が与えられる。公差 1 の等差数列（＝連続する整数の列、負の整数も含めてよい）であって、総和が $N$ になるものは何通りあるか。
- 制約: $1 \le N \le 10^{12}$（$N$ は整数）。$N$ を全部なめる解法は重い、という伏線。
- 入出力例（公式サンプル 1）: $N=12$ → **4**。内訳は $[12]$, $[3,4,5]$, $[-2,\dots,5]$, $[-11,\dots,12]$ の 4 つ。これが「何を数えるか」の具体像で、後ろの要点③の対応表とそのまま対応する。

## 考察

### 要点①: 連続整数列の和を公式から書き下す

- 自分はまず「連続整数列の和」を、$1$ から $n$ までの和の差として書いた:
  $$\frac{n(n+1)}{2} - \frac{m(m+1)}{2} = N$$
- これは数列 $[m+1, m+2, \dots, n]$ の和。この式が **2 つの操作** を同時に表しているのが面白かった:
  - 列 $[1, 2, \dots, m, \dots, n]$ から $[1, 2, \dots, m]$ を **取り除く** 操作
  - 列 $[1, 2, \dots, n]$ に $[-m, -(m-1), \dots, -1, 0]$ を **付け足す** 操作（負の項で帳尻が合う）
- どちらも同じ値 $\frac{n(n+1)}{2} - \frac{m(m+1)}{2}$ になる。具体例で見ると、$N=12$・$n=5,\,m=2$ のとき:
  - **取り除き**: $[1,2,3,4,5]$ から $[1,2]$ を抜いて $[3,4,5]$（$15-3=12$）
  - **付け足し**: $[1,2,3,4,5]$ に $[-2,-1,0]$ を足して $[-2,-1,0,1,2,3,4,5]$（$15-3=12$）

![同じ「15 − 3 = 12」が、先頭を取り除く操作と先頭に負の項を付け足す操作の 2 通りの数列を生む様子](/images/20260618-atcoder-abc190-d-staircase-sequences/remove-add.png)

- どちらも和は $12$ だが、出てくる数列は別物。**1 つの式が 2 本の数列を生む** この感覚が、最後の答えの **×2** に直結する（要点③で回収する）。

### 要点②: 素朴解法（n を回して m を二分探索）と限界

- 最初の方針: $n$ をカウントアップしながら、条件を満たす $m$ を **二分探索** で探す。
- なぜループが重いか: $n$ を 1 から順に試していくと、最短の数列は単独項 $[N]$（$n=N$）なので、候補の $n$ は最大 $N$ まである。つまり $O(N)$ 本のループになり、各 $n$ で $m$ を二分探索（$O(\log N)$）すると全体で $O(N \log N)$。
- $N \le 10^{12}$ では $N\log N \approx 4\times 10^{13}$ で、これは間に合わない（TLE）。「ループの**本数そのもの**を $O(\sqrt{N})$ に減らせないか」——この不満が、次の式変形（積の形 → 約数列挙）の動機になる。

### 要点③: 積に直すと、数列の長さが 2N の約数になる

- 最初の式を整理する:
  $$\frac{n(n+1)}{2} - \frac{m(m+1)}{2} = N$$
  $$n(n+1) - m(m+1) = 2N$$
  $$(n - m)(n + m + 1) = 2N$$
- ここで 2 つの因数 $n - m$ と $n + m + 1$ に注目すると、和が $(n-m) + (n+m+1) = 2n + 1$ で **奇数** → 2 因数の **偶奇は必ず異なる**。
- つまり $2N$ を「偶奇の異なる 2 因数の積」に分解する通り数を数えればよく、これは **約数列挙**（$O(\sqrt{N})$）でできる。
  - 各分解で 2 因数を $n-m$ と $n+m+1$ に対応させると、$n = \frac{(n-m) + (n+m+1) - 1}{2}$, $m = \frac{(n+m+1) - (n-m) - 1}{2}$ として $n, m$ が整数で復元できる。
- 結論: **答え = $N$ の奇約数の個数 $\times\,2$**。

「×2」の出どころは、要点①で見た二重性そのもの。**偶奇の異なる 2 因数の組 1 つ**から、長さ $n-m$ の数列（取り除き版）と長さ $n+m+1$ の数列（負への拡張版）の **2 本**が立つ。$N=12$ で全部並べるとこうなる:

| 因数の組 $(n-m,\ n+m+1)$ | 取り除き版（長さ $n-m$） | 負への拡張版（長さ $n+m+1$） |
|---|---|---|
| $(1,\ 24)$ | $[12]$ | $[-11,\dots,12]$ |
| $(3,\ 8)$ | $[3,4,5]$ | $[-2,\dots,5]$ |

負への拡張版は、取り除き版の頭に **総和 0 の対称ブロック**（$[3,4,5]$ なら $[-2,-1,0,1,2]$）を足したもの。要するに、**1 つの因数の組から、長さ $n-m$（取り除き）と長さ $n+m+1$（負への拡張）の 2 本を数え上げている**。因数の組は奇約数の個数だけあるので、**答え = 奇約数の個数 × 2**。検算: 奇約数 $\{1,3\}$ の 2 個 → $2\times2=4$（サンプル一致）。

## 解法・実装

- 実装方針（手順）:
  1. $2N$ の約数を $O(\sqrt{N})$ で列挙する
  2. 各約数ペアのうち偶奇が異なるものだけ数える（＝有効な数列に対応）
  3. その個数を出力する
- 自分の提出（Python、フルは [こちら](https://atcoder.jp/contests/abc190/submissions/76760726)）。要点が伝わるよう少し整理した抜粋:

```python
N = int(input())

ans = 0
M = 2 * N                      # M = 2N
for i in range(1, M + 1):
    if i * i > M:              # i は √(2N) まで → O(√N)
        break
    if M % i != 0:
        continue

    d1 = i                     # d1 = n - m（取り除き版の数列の長さ）
    d2 = M // i                # d2 = n + m + 1（負への拡張版の数列の長さ）

    if d1 % 2 == d2 % 2:       # 偶奇が同じ → 有効な数列に対応しないので捨てる
        continue

    ans += 2                   # 偶奇が異なる分解 1 つにつき、取り除き版＋負への拡張版で 2 つ

print(ans)
```

- 変数の対応: `d1 = i` が $n - m$（＝**取り除き版の数列の長さ**）、`d2 = M // i` が $n + m + 1$（＝**負への拡張版の数列の長さ**）。$2N$ の 2 因数が、そのまま双子の数列それぞれの長さになっている。`i` は $\sqrt{2N}$ まで回すので $O(\sqrt{N})$。
- `d1 % 2 == d2 % 2`（偶奇が同じ）を捨てるのが「**偶奇の異なる 2 因数の積**」の条件そのもの。残った各ペアが有効な数列に対応する。
- ループ内の `ans += 2`: `i` を $\sqrt{2N}$ までしか回さないので 1 周で見つかるのは**非順序ペア 1 つ（＝奇約数 1 個）**。そこに「取り除き版＋双子（負への拡張版）」で $+2$ するのが、要点③で見た ×2 そのもの。**この `+= 2` がこの記事の山場**（二重性の回収点）になっている。
- 計算量: 約数列挙で $O(\sqrt{N})$。$N \le 10^{12}$ でも $\sqrt{N} \approx 10^6$ で間に合う。没にした素朴解法 $O(N \log N)$（TLE）との対比で、約数列挙の速さが際立つ。

## この問題の面白いところ

- 「**数列の和** を数える問題」が、式変形を経て「**約数を列挙する** 問題」に化けるのが面白い。最初の見た目からずいぶん遠くまで来たな、という感覚。
- 同じ式 $\frac{n(n+1)}{2} - \frac{m(m+1)}{2}$ が「列を削る／負の項を足す」の二通りに読める、その二重性も気持ちよかった。
- 発想の汎用性: 「和・差の形を積の形に変形して約数に帰着」は、他の数え上げでも効く考え方だと思う。
